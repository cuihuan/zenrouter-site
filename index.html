常见问题

我购买的是什么？

ZenRouter 服务的预付 API 额度。支付成功后立即到账，调用 API 时按所用模型的 token 单价逐次扣减。每次购买都是一次性的——没有订阅，绝不自动续费。

支持哪些支付方式？

支持国际信用卡、PayPal 等方式在线支付，更多本地支付方式陆续开通中。所有支付由持牌支付平台处理，我们不接触你的卡片信息。

退款政策是什么？

未使用的额度自购买起 7 天内可退，详见退款政策。

额度会过期吗？

自购买之日起 12 个月内有效，到期前我们会邮件提醒。

和直接用官方 API 有什么区别？

一个 Key 管理所有厂商的模型、统一余额与账单、多上游自动故障转移；接口与官方 SDK 完全兼容，迁移只需改 base URL。

遇到问题怎么找到你们？

邮件 support@zenrouter.io，账单与退款类问题 2 个工作日内回复。
