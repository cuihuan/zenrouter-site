Up and running in two minutes

Works with the tools you already use — just change the base URL. 两分钟接入，只需替换 base URL。

# ~/.zshrc or ~/.bashrc
export ANTHROPIC_BASE_URL=https://api.zenrouter.io
export ANTHROPIC_AUTH_TOKEN=your-zenrouter-key

claude   # start Claude Code as usual
  1. Create an API key in your dashboard
  2. Export the two environment variables above
  3. Launch Claude Code — requests now route through ZenRouter
# ~/.codex/config.toml
[model_providers.zenrouter]
name = "ZenRouter"
base_url = "https://api.zenrouter.io/v1"
env_key = "ZENROUTER_API_KEY"
  1. Add the provider block to your Codex config
  2. Run export ZENROUTER_API_KEY=your-key

# pip install openai

    base_url="https://api.zenrouter.io/v1",
    api_key="your-zenrouter-key",
)

    model="gpt-5.5",
    messages=[{"role": "user", "content": "Hello"}],
)

Frequently asked questions

What exactly am I buying?

Prepaid API credits for the ZenRouter service. Credits are added to your account immediately after payment and are consumed as you make API requests, priced per token by model. Each purchase is one-time — there is no subscription and no recurring charge.

Is this a subscription?

No. You are never charged automatically. When your balance runs low you can choose to buy more credits — entirely at your discretion.

What is the refund policy?

We offer a 7-day refund for unused credits. See the full Refund Policy for details.

Do credits expire?

Credits remain valid for 12 months from the date of purchase. We notify you by email before any credits are due to expire.

Which models are available?

We aggregate leading models from multiple providers behind one OpenAI-compatible API — GPT-series, Claude-series, Gemini-series and popular open-weight models. The live list and per-token rates are in your dashboard.

How do I get help?

Email support@zenrouter.io. We respond to all billing and refund inquiries within 2 business days.
